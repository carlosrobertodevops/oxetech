# Plano: Gerenciador de Tarefas (AV1 Frontend)

## Requisitos
- HTML Semântico (`header`, `main`, `footer`, `section`).
- CSS Framework: Tailwind CSS (via CDN).
- Estilo: Flexbox, Grid, Responsivo (`media query`, `flex-wrap`).
- JS: Manipulação de DOM (Adicionar, remover, validar itens).

## Estrutura
- `index.html`: UI principal.
- `css/style.css`: Estilos customizados (se necessário).
- `js/app.js`: Lógica de tarefas.
- `README.md`: Documentação.

## Delegação (Subagentes)
1. HTML (index.html).
2. Lógica (js/app.js).
3. Docs (README.md).
